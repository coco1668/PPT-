# ppt_clock
在deepin系统上制作的PPT倒计时工具
这个PPT倒计时工具并不完整，功能上还是有缺陷的。
它可以完成
: 1.总时间计时；
: 2.倒计时前多少分钟进行声音播放提醒；
: 3.倒计时开始前多少秒进行倒计时；
: 4.倒计时结束后是否播放提示音；
: 5.倒计时结束后，是否继续播放PPT，还是关闭PPT；
: 6.倒计时结束后是否继续计时；
